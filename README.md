## Notebooks

This repo is organised around the three sequential questions the analysis answers. Each notebook is self-contained and corresponds to one stage of the pathway.

**`Question_1.ipynb`**
*Does education and exposure to information predict contraceptive use, as a marker of reproductive empowerment?*
Dataset: Contraceptive Method Choice (CMC), UCI
Method: Binary logistic regression, 80/20 train–test split, ROC–AUC

**`Question_2.ipynb`**
*Do empowerment and information access predict behavioural risk relevant to cervical cancer?*
Dataset: Cervical Cancer Behavioural Risk (SOBAR), UCI
Method: Hierarchical linear regression (2-block)

**`Question_3.ipynb`**
*Do behavioural risk differences translate into clinical cervical cancer outcomes?*
Dataset: Cervical Cancer Risk Factors, UCI
Method: Binary logistic regression with interaction term, 80/20 train–test split, sensitivity analysis

Read them in order — each stage's findings motivate the modelling choice in the next.
